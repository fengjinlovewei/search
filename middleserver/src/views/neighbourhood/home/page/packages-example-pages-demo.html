# Typescript 使用指南

# 说明

  1. 本次升级同时支持 js 和 ts,开发时会在编译阶段对当前启动目录（如果有 ts 文件）则会进行静态类型检查
  2. 同时 eslint 也会进行代码风格规范，vscodo 安装插件 ESLint, Prettier
  3. 配置文件为 ts.config.json,新增目录或者 alias 无法解析时，需要在里面配置
  4. example 文件为 ts+hooks 开发，如需新增其他 demo 文件可在 example 中配置

# 类型定义速查

  1. React.FC | React.FunctionComponent 函数组件类型

  2. React.Component<Props, State> class 组件类型

  3. React.ComponentType 表示 (React.FC | React.Component) 集合的类型 - 用于 HOC

  4. React.ComponentProps 取得组件 XXX 的 Props 类型（警告：无法用于静态声明的 default props 以及泛型 props）

  5. React.ReactElement | JSX.Element 表示 React 中 Element 概念的类型 - 表示一个原生 DOM 组件（比如

    ）或用户自定义的复合组件 （比如 ）

  6. React.ReactNode 表示任意类型的 React 节点（相当于 ReactElement (包括 Fragments 和 Portals) + 原始 JS 类型）

  7. React.CSSProperties 表示 JSX 中样式对象的类型

  8. React.HTMLProps 表示指定 HTML 元素的类型 - 用于扩展 HTML 元素

  9. React.ReactEventHandler 表示 event handler 的泛型类型 - 用于声明 event handlers

  10. React.XXXEvent 表示更多特殊 event。一些常见的 event 例如：ChangeEvent, FormEvent, FocusEvent, KeyboardEvent, MouseEvent, DragEvent, PointerEvent, WheelEvent, TouchEvent。
